# 原型蓝图 — Apex Drive Store V2

> **这是什么**：深色座舱（Dark Cockpit）静态界面（HTML/CSS，纯视觉无逻辑）。浏览器开 [index.html](index.html) 为索引页，双击任意磁贴查看单页。

> - 后台 A0–A5：`admin-login` `admin-products` `admin-banners` `admin-services` `admin-orders` `admin-session`
> - 共享样式：[ui.css](ui.css)（:root 令牌 + 全部组件，落自 design-tokens.md）；组件与字段绑定见 [../component-spec.md](../component-spec.md)。
> **用途**：作为高保真原型与前端实现的依据。
> 视觉取值见 [../design-tokens.md](../design-tokens.md)，文案见 [../../product/prd.md](../../product/prd.md) §11，字面值见 [../../product/prd.md](../../product/prd.md) §10。

## Admin（六模块，管理后台）
- **/admin/login**：admin/123456 登录；错误提示复用 COPY-003。
- **/admin/products**：商品管理，列表/新增/编辑/上下架/库存/排序；实物与会员均可管理。
- **/admin/banners**：Banner 管理，列表/新增/编辑/上下架/跳转目标/排序。
- **/admin/services**：服务内容管理，会员权益/价格/详情；展示服务说明/上下架。
- **/admin/orders**：订单管理，只读列表/详情/类型筛选/状态筛选。
- **/admin/session**：演示会话管理，查看 auth/drive/net/cart/checkout/order/membership；可选重置。

Admin 是 v1 范围内的真实管理后台，不是只读原型。字段和保存规则以 PRD §15.5、§15.11 为准；具体视觉可在后续高保真中细化。

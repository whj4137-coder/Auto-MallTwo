# 演示脚本 — Apex Drive Store V2（5–10 分钟）

> 面向内部评审的可运行演示流程。演示四件事：驻车低干扰购物闭环、行车/断网安全门禁、三类商品差异化、前台/Admin/后端三端联动。
> 启动见 [../engineering/dev-guide.md](../engineering/dev-guide.md) §2：`npm run dev:server` + `npm run dev:web`。前台 http://localhost:5173/ ，后台 http://localhost:5173/admin （admin/123456）。浏览器 1280×720。

## 0. 开场（30s）
- 一句话：车机横屏商城概念 Demo，**不接真实支付**；演示驻车购物闭环 + 安全门禁 + 三端联动。
- 顶部 **Demo Bar** 三组开关（驻车/行车 · 在线/断网 · 已登录/未登录）+「重置数据」。强调：三态**只手动切换，绝不自动触发**。
- 先点「重置数据」回到干净状态（GUEST / PARKED / ONLINE）。

## 1. A-01 实物购物闭环（2 min）
1. 首页（P1）：Bento 主推 + 类目条 + 精选横滑 + 分类货架；指出商品图。
2. 点「磁吸手机支架」→ 详情 P3（SKU 颜色、数量 1–5）。
3. 点「加入购物车」→ **GUEST 弹登录弹窗**（演示账号已预填 admin/123456）→ 登录后**自动续作**加购，toast「已加入购物车」。
4. SideNav→购物车 P6：勾选/改量/价格/删除 + 摘要（收货人取自 /me）。
5. 前往结算 P7 → 去支付 P8（二维码占位 +「演示支付，不会产生真实扣款」）→「模拟扫码成功」。
6. 支付结果 P9：**支付成功 + ORDER-P-001 + 已支付** → 查看订单 P10 → 点进订单详情 P13。

## 2. B-01 会员开通（1 min）
1. 首页/类目进「悦行会员月卡」P4：渐变会员卡 + 权益 + 绑定车辆。
2. 立即开通 → 确认开通 P7 → 模拟扫码成功 → **权益已开通 + ORDER-M-001**。
3. 回会员页：两张会员卡按钮均变「已开通 · 查看订单」（**全局单一、月/年卡互斥、幂等**）。「我的」P11 会员状态=已开通。

## 3. 安全门禁（1.5 min）
- 切 **行车**：商品详情「加入购物车」**置灰**，点击 toast「行车中仅支持浏览…」，**不发请求**。
- 切回驻车、切 **断网**：同样置灰 + toast「当前网络不可用…」。
- 强调优先级 **行车 > 断网 > 未登录**（同时命中只显最高）。
- 展示服务（C-01）：进「城市快充服务」P5 → 只有「暂未开放」灰按钮，**无加购/购买入口**（强制写入后端返回 2003）。

## 4. 三端联动 · Admin（2 min）
1. 开 http://localhost:5173/admin （admin/123456）。六模块：商品/Banner/服务/订单/演示会话/账号信息。
2. **商品管理**：把「磁吸手机支架」**下架**（开关）→ 回前台首页/分类/搜索**该商品立即消失**；购物车若有它，结算被拦（COPY-045）。
3. 改价或**新增商品** → 前台实时可见（演示「Admin 改、前台读」）。
4. **订单管理**：可见刚才的 ORDER-P-001 / ORDER-M-001。
5. **演示会话**：运行快照（购物车/订单/会员/序号）+ 重置。

## 5. 搜索 + 收尾（1 min）
- 首页搜索框 → /search，输「支架」命中；输无关词→「未查询到你想要的结果」。
- 点「重置数据」：清会话、订单号回 001、回首页 GUEST/PARKED/ONLINE（**保留** Admin 内容编辑）。

## 备注
- 金额一律 `¥整数`、无运费/税/优惠；订单号 `ORDER-P/M-NNN` 独立递增、重置回 001。
- 自动化测试：`npm run verify`（L1/L2，68）/ `npm run test:e2e`（L3，10）。
- 仅深色座舱、仅简体中文、固定 1280×720。
